# 澜起科技假研报识别模式

> entity_type: investment_stock

**类型:** Pattern  
**ID:** `pattern_lanqi_fake_report`  
**来源:** 假研报辟谣分析（2026-05-14）

---

## 识别特征

| 特征 | 描述 |
|------|------|
| **日期超前** | 文档日期为未来日期（如2026年5月13日） |
| **数据夸张** | 92%市占、精确到百万的净利润预测 |
| **无来源** | 没有机构署名、没有官方公告 |
| **反常识** | 与基本面/行业常识明显矛盾 |
| **绝对化用语** | "独家供应商"、"焊死护城河"等 |

---

## 本案实例

2026年5月14日群聊中出现一张"澜起科技研报截图"，声称：
- 三星CMM-D模块将**独家使用**澜起MXC芯片
- 市占率**92%**
- 苹果将**大量采购Intel Xeon服务器芯片**

**辟谣分析：**
1. 苹果花了十年摆脱Intel转自研芯片，回头大量买Xeon？**反常识**
2. 92%市占、精确净利润，无任何机构署名 → **出货文特征**
3. 日期显示"2026年5月13日" → 用来规避追责的虚假时间戳

---

## 教训

群里传播的「出货文」特征：
- 时间超前 + 数据夸张 + 无来源 + 反常识
