# Federal Reserve（美联储）

美国中央银行，负责制定货币政策（联邦基金利率）。2026年6月FOMC会议显示鹰派转向，年内可能再加息一次25bp。

## 关键人物
- 主席：沃什（Kevin Walsh），2026年新任，最年轻理事（35岁），删除前瞻指引，设立5个工作组改革美联储

## 2026年6月会议关键数据
- 联邦基金利率：3.5%-3.75%（连续4次按兵不动）
- PCE通胀预测：2.7% → 3.6%（大幅上调）
- GDP预测：2.4% → 2.2%（下调）
- 点阵图中值：年底3.8%（暗示加息1次）

## 对中国市场影响
- 人民币汇率承压（中美利差倒挂持续）
- A股外资配置盘流出（但传导有限，国内政策可对冲）
- 港股估值受压（港币联系汇率直接传导）
- 中国央行降息空间被压缩
