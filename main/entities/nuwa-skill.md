# nuwa-skill

**把人的思维方式、表达 DNA 蒸馏成可调用的 Agent Skills。**

GitHub: [alchaincyf/nuwa-skill](https://github.com/alchaincyf/nuwa-skill)

## 使命

蒸馏"人"——把一个高价值个体的思维方式、决策风格、表达 DNA 提取出来，固化成 AI agent 可调用的 skill。即使这个人离职了，他的经验、语气、工作方式也能被 AI 一定程度替代。

## 核心能力

- **人类 skill 创造**：生成基于特定人物的 skill 包，如"马斯克 skill"、"巴菲特 skill"
- **思维模式提取**：从公开资料中提取核心思维框架、决策习惯、表达风格
- **风格还原**：不仅还原逻辑，还还原表达的语气、节奏、比喻习惯

## 典型产出

人物级 skill 包，包含：
- 核心思维模型（如巴菲特的"能力圈"、"护城河"）
- 决策框架与原则
- 表达方式与语言风格模板
- 知识边界与盲点

## 生态定位

nuwa-skill 是更大 skill 生态的一部分：
- **nuwa-skill**（本仓库）— 蒸馏人（思维方式、表达 DNA）
- **[cangjie-skill](https://github.com/kangarooking/cangjie-skill)** — 蒸馏书/视频（方法论、框架、原则）
- **[darwin-skill](https://github.com/alchaincyf/darwin-skill)** — 进化任意 skill

三者咬合：nuwa 蒸馏人，cangjie 蒸馏书，darwin 让它们持续进化。
